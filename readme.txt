Show an elegant ad below content area.

== Description ==
An elegant ad below content area. With this plugin you can add title, description, link, image etc.
[DEMO LINK](https://qdonow.com/insights/5-best-online-proofreading-tools-for-bloggers-and-writers/)

Features::
- Fields: Title, Description, Link, Image
- Options: Sponsor tag, Dark mode
- Support all post types

== Installation ==
1. Install the plugin
1. Go to WordPress dashboard > Plugins > Installed Plugins
1. Activate the ‘Kantbtrue Content Bottom ADS’
1. Go to ‘KBTCBA’ from admin menu and edit as per your needs

== Frequently Asked Questions ==
= How to add custom CSS? =
To add custom CSS, you can use the WordPress default 'Additional CSS' section from the customizer.
CSS class naming list:
- container: kbtcba-footer-ad
